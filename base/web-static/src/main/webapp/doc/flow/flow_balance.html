客户余额查询接口

功能

本接口是客户余额查询

调用方向

亿美软通平台 >>客户服务端

地址

http://ip:port/interface/balance

结果

json 格式

HTTP METHOD

POST

参数

参数 类型 是否必填 描述
appIdString是我方唯一标识
signString是32位Md5签名方式，token为我司提供密钥；
sign = md5(appId= appId+token)

示例

appId=xx&sign=xx

响应

响应数据格式为Json

参数

参数 类型 描述
balanceString该appId余额
creditString该appId授信，即可以透支的金额
responseCodeString查询状态码，SUCCESS代表查询成功

示例

  • {
    "balance": "xxxx",
    "credit": "xxxx",
    "responseCode": "SUCCESS"
}

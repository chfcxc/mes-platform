提交流量充值订单

功能

本接口是api接口充值接口

调用方向

亿美软通平台 >>客户服务端

地址

http://ip:port/interface/order

返回

订购结果 json串

HTTP METHOD

POST

参数

参数 类型 是否必填 描述
appIdString是我方唯一标识
signString是32位Md5签名方式，token为我司提供密钥；
sign = md5(appId=appId&mobile=mobile&clientTaskNo=clientTaskNo&productCode=productCode+token)
clientTaskNoString否由客户产生的此订单编号，不能超过64位
mobileString是终端的电话号码,如13801234567
productCodeString是我方流量编码，如M5，T1024，U20；
采用英文缩写+包型大小的方式生成；
编码生成策略：M代表中国移动，T代表中国电信，U代表中国联通，数字是流量包型大小；
10M是10，1g是1024，以此类推

示例

appId=xx&mobile=xxx&clientTaskNo=xx&productCode=T5&sign=xx

响应

响应数据格式为Json

参数

参数 类型 描述
clientTaskNoString由客户产生的此订单编号，不能超过64位
responseCodeString此次请求的状态码，SUCCESS是接收成功，其余均为接收不成功；
如果请求成功,必然生成我方订单号，平台单号的产生,并不意味充值成功,只表明平台已接收请求；
当responseCode等于SUCCESS的时候，表示下单成功，其他情况下根据不同情况是不同的错误码
taskNoString我司订单编号，用来查询订单状态

示例

  • {
    "clientTaskNo": "a4fd8ab7-249a-431e-be93-12b0c4ca3cc9",
    "responseCode": "SUCCESS",
    "taskNo": "xxxx"
}

充值结果回调接口

功能

本接口是api接口充值接口

调用方向

亿美软通平台 >>客户服务端

调用地址

客户提供的地址，绑定在服务号上

HTTP METHOD

POST

参数

参数 类型 描述
clientTaskNoString客户订单号
taskNoString我司订单号
codeString充值状态码
只有两种状态，SUCCESS是充值成功，其余均为失败

示例

  • {
    "clientTaskNo": "xxxx",
    "code": "SUCCESS",
    "taskNo": "xxxx"
}
  • 客户的WEB服务端在接收到回调后,请返回字符串 SUCCESS表示接收成功。
  • 如客户返回的字符串不是这7个字符,则平台视为回调不成功,将在一定时间后重新回调一定的次数。
  • 第一次回调时间:我方收到运营商状态之后,会马上推送给客户的系统。
  • 如果第一次推送不成功，将会在10分钟内再推送一次，如果还不成功，将会在20分钟内再推送一次。三次均推送后，我方认为推送完成，不再进行推送。如果长时间没收到状态报告，请联系客服人员重新推送。

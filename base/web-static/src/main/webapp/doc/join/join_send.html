金融营销接口

  金融营销接口，所有参数均通过post参数提交,注意：部分参数如内容需要 进行URLENCODE转码保证数据能够正常传输。

字符集：普通接口传出数据仅支持UTF-8字符集编码；

发送营销接口

请求

请求数据格式为post

链接

http://ip:port/join/send

参数

参数 类型 描述
appId String 用户Appid(必填)
timestamp String 时间戳(必填,提交时间)
格式：yyyyMMddHHmmss 14位（保证跟北京
时间误差1分钟之内）
sign String 签名(必填)
格式：md5(appId+ secretKey + timestamp)
32位
voiceContent String 语音内容(必填，语音内容+互动语音内容+
留言语音内容 最多500字）
customJoinIds String 自定义消息ID(选填)最长64位
与手机号一一对应 最多500个，使用英文逗号分
隔
isInteractive String (选填)是否互动 0 不互动 , 1 互动（根据业务填
写）
interactiveContent String (选填)互动语音内容(选填，语音内容+互动语音内
容+留言语音内容 最多500字）
isAcousticSign String (选填)是否留言 0 不留言 , 1 留言（根据业务填
写）
acousticSignContent String (选填)留言语音内容(选填，语音内容+互动语音内
容+留言语音内容 最多500字）
smsContent String (选填)短信内容
triggerConditions String (选填)短信触发条件，1-接通时，2-未接通，
3-按键）4-接通时间到达配置时间时
keyContent String (选填)按键触发短信按键（数字0-9， 当
triggerConditions 为 3 时必填）
connectTime String (选填)接通时间到达配置时间,按键为4时必填
timingTime String (选填)定时时间，格式为“yyyyMMddHHmmss

示例

http://ip:port/join/send?appId=EUCP-EMY-DDDD-3EEEE&timestamp=201701011200
00&sign=PIEUDJI987EUID62PKEDSESQEDSEDFSE&mobiles=18001098901&voiceContent=1234&cust
omJoinIds=10001&isInteractive=1&interactiveContent=1234&acousticSignContent=nihao&s
msContent=nihao&triggerConditions=3&keyContent=1&timingTime=20180809101010&connectTime=10

响应

响应数据格式为Json。

参数

参数 类型 描述
code String 请求状态码，详见附录《接口状态码表》
mobile String 手机号
joinId String 消息ID
customJoinId String 客户自定义消息ID
